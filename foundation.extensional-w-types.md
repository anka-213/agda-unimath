## Idea

A W-type `ùïé A B` is said to be extensional if for any two elements `S T : ùïé A B` the induced map

```md
  Id S T ‚Üí ((U : ùïé A B) ‚Üí (U ‚àà-ùïé S) ‚âÉ (U ‚àà-ùïé T))
```

is an equivalence.
