Idea

Since we sometimes think of types as ‚àû-groupoids, with the groupoid structure provided implicitly by the identity type and its induction principle, we can think of maps as functors of ‚àû-groupoids. We borrow some terminology of functors, and call a map faithful if it induces embeddings on identity types.

Definition

module _
  {l1 l2 : Level} {A : UU l1} {B : UU l2}
  where

  is-faithful : (A ‚Üí B) ‚Üí UU (l1 ‚äî l2)
  is-faithful f = (x y : A) ‚Üí is-emb (ap f {x} {y})

faithful-map : {l1 l2 : Level} ‚Üí UU l1 ‚Üí UU l2 ‚Üí UU (l1 ‚äî l2)
faithful-map A B = Œ£ (A ‚Üí B) is-faithful

module _
  {l1 l2 : Level} {A : UU l1} {B : UU l2}
  where

  map-faithful-map : faithful-map A B ‚Üí A ‚Üí B
  map-faithful-map = pr1

  is-faithful-map-faithful-map :
    (f : faithful-map A B) ‚Üí is-faithful (map-faithful-map f)
  is-faithful-map-faithful-map = pr2

  emb-ap-faithful-map :
    (f : faithful-map A B) {x y : A} ‚Üí
    Id x y ‚Ü™ Id (map-faithful-map f x) (map-faithful-map f y)
  pr1 (emb-ap-faithful-map f {x} {y}) = ap (map-faithful-map f)
  pr2 (emb-ap-faithful-map f {x} {y}) = is-faithful-map-faithful-map f x y

  is-faithful-is-emb : {f : A ‚Üí B} ‚Üí is-emb f ‚Üí is-faithful f
  is-faithful-is-emb {f} H x y = is-emb-is-equiv (H x y)

  faithful-map-emb : (A ‚Ü™ B) ‚Üí faithful-map A B
  pr1 (faithful-map-emb f) = map-emb f
  pr2 (faithful-map-emb f) = is-faithful-is-emb (is-emb-map-emb f)

  is-faithful-is-equiv : {f : A ‚Üí B} ‚Üí is-equiv f ‚Üí is-faithful f
  is-faithful-is-equiv H = is-faithful-is-emb (is-emb-is-equiv H)

  faithful-map-equiv : (A ‚âÉ B) ‚Üí faithful-map A B
  pr1 (faithful-map-equiv e) = map-equiv e
  pr2 (faithful-map-equiv e) = is-faithful-is-equiv (is-equiv-map-equiv e)

  emb-ap : (f : A ‚Ü™ B) (x y : A) ‚Üí Id x y ‚Ü™ Id (map-emb f x) (map-emb f y)
  pr1 (emb-ap f x y) = ap (map-emb f) {x} {y}
  pr2 (emb-ap f x y) = is-faithful-is-emb (is-emb-map-emb f) x y

Examples

The identity map is faithful

module _
  {l : Level} {A : UU l}
  where
  
  id-faithful-map : faithful-map A A
  id-faithful-map = faithful-map-emb id-emb

  is-faithful-id-faithful-map : is-faithful (id {A = A})
  is-faithful-id-faithful-map = is-faithful-map-faithful-map id-faithful-map

Any 0-map is faithful

Properties

The projection map of a family of sets is faithful

Faithful maps are closed under homotopies

module _
  {l1 l2 : Level} {A : UU l1} {B : UU l2} {f g : A ‚Üí B} (H : f ~ g)
  where

  abstract
    is-faithful-htpy : is-faithful g ‚Üí is-faithful f
    is-faithful-htpy K =
      is-faithful-is-0-map (is-0-map-htpy H (is-0-map-is-faithful K))

Faithful maps are closed under composition

module _
  {l1 l2 l3 : Level} {A : UU l1} {B : UU l2} {X : UU l3}
  (f : A ‚Üí X) (g : B ‚Üí X) (h : A ‚Üí B) (H : f ~ (g ‚àò h))

  abstract
    is-faithful-comp : is-faithful g ‚Üí is-faithful h ‚Üí is-faithful f

          ( is-0-map-is-faithful K)
          ( is-0-map-is-faithful L))
