Definition

Maps f : A ‚Üí B of which the fibers are sets, i.e., 0-truncated types, are called 0-maps. We will show in foundation-core.faithful-maps that a map f is a 0-map if and only if f is faithful, i.e., f induces embeddings on identity types.

module _
  {l1 l2 : Level}
  where

  is-0-map : {A : UU l1} {B : UU l2} ‚Üí (A ‚Üí B) ‚Üí UU (l1 ‚äî l2)
  is-0-map {A} {B} f = (y : B) ‚Üí is-set (fib f y)

  0-map : (A : UU l1) (B : UU l2) ‚Üí UU (l1 ‚äî l2)
  0-map A B = Œ£ (A ‚Üí B) is-0-map

  map-0-map : {A : UU l1} {B : UU l2} ‚Üí 0-map A B ‚Üí A ‚Üí B
  map-0-map = pr1

  is-0-map-map-0-map :
    {A : UU l1} {B : UU l2} (f : 0-map A B) ‚Üí is-0-map (map-0-map f)
  is-0-map-map-0-map = pr2

Properties

Projections of families of sets are 0-maps

For any type family over the codomain, a 0-map induces a 0-map on total spaces.

In other words, 0-maps are stable under pullbacks. We will come to this point when we introduce homotopy pullbacks.
