Idea

We say that a map f : A ‚Üí B into a k-truncated type B is a k-truncation of A ‚Äì or that it satisfies the universal property of the k-truncation of A ‚Äì if any map g : A ‚Üí C into a k-truncated type C extends uniquely along f to a map B ‚Üí C.
