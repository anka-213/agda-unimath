Idea

The k-truncation image of a map f : A ‚Üí B is the type trunc-im k f that fits in the (k-connected,k-truncated) factorization of f. It is defined as the type

  trunc-im k f := Œ£ (y : B), type-trunc k (fib f y)

Definition

module _
  {l1 l2 : Level} (k : ùïã) {A : UU l1} {B : UU l2} (f : A ‚Üí B)
  where
  
  trunc-im : UU (l1 ‚äî l2)
  trunc-im = Œ£ B (Œª y ‚Üí type-trunc k (fib f y))

  unit-trunc-im : A ‚Üí trunc-im
  pr1 (unit-trunc-im x) = f x
  pr2 (unit-trunc-im x) = unit-trunc (pair x refl)

  projection-trunc-im : trunc-im ‚Üí B
  projection-trunc-im = pr1
