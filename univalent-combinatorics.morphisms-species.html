Idea

A homomorphism between two species is a pointwise family of maps between their values.

Definitions

Morphisms of species

hom-species :
  {l1 l2 l3 : Level} ‚Üí
  species l1 l2 ‚Üí species l1 l3 ‚Üí UU (lsuc l1 ‚äî l2 ‚äî l3)
hom-species {l1} F G = (X : ùîΩ l1) ‚Üí F X ‚Üí G X

id-hom-species : {l1 l2 : Level} ‚Üí (F : species l1 l2) ‚Üí hom-species F F
id-hom-species F = Œª X x ‚Üí x 

comp-hom-species :
  {l1 l2 l3 l4 : Level}
  {F : species l1 l2} {G : species l1 l3} {H : species l1 l4} ‚Üí
  hom-species G H ‚Üí hom-species F G ‚Üí hom-species F H
comp-hom-species f g X = (f X) ‚àò (g X)

Homotopies between morphisms of species

htpy-hom-species :
  {l1 l2 l3 : Level} {F : species l1 l2} {G : species l1 l3} ‚Üí
  hom-species F G ‚Üí hom-species F G ‚Üí UU (lsuc l1 ‚äî l2 ‚äî l3)
htpy-hom-species {l1} f g = (X : ùîΩ l1) ‚Üí (f X) ~ (g X)

refl-htpy-hom-species :
  {l1 l2 l3 : Level} {F : species l1 l2} {G : species l1 l3}
  (f : hom-species F G) ‚Üí htpy-hom-species f f
refl-htpy-hom-species f X = refl-htpy

Properties

Homotopies characterize the identity type of morphisms of species

htpy-eq-hom-species :
  {l1 l2 l3 : Level} {F : species l1 l2} {G : species l1 l3}
  {f g : hom-species F G} ‚Üí
  Id f g ‚Üí htpy-hom-species f g
htpy-eq-hom-species refl X y = refl

is-contr-htpy-hom-species :
  {l1 l2 l3 : Level} {F : species l1 l2} {G : species l1 l3}
  (f : hom-species F G) ‚Üí is-contr (Œ£ (hom-species F G) (htpy-hom-species f))
is-contr-htpy-hom-species f =
  is-contr-total-Eq-Œ† (Œª X h ‚Üí f X ~ h) (Œª X ‚Üí is-contr-total-htpy (f X))

is-equiv-htpy-eq-hom-species :
  {l1 l2 l3 : Level} {F : species l1 l2} {G : species l1 l3}
  (f g : hom-species F G) ‚Üí is-equiv (htpy-eq-hom-species {f = f} {g = g})
is-equiv-htpy-eq-hom-species f =
  fundamental-theorem-id
    ( is-contr-htpy-hom-species f)
    ( Œª g ‚Üí htpy-eq-hom-species {f = f} {g = g})

eq-htpy-hom-species :
  {l1 l2 l3 : Level} {F : species l1 l2} {G : species l1 l3}
  {f g : hom-species F G} ‚Üí htpy-hom-species f g ‚Üí Id f g 
eq-htpy-hom-species {f = f} {g = g} =
  map-inv-is-equiv (is-equiv-htpy-eq-hom-species f g)

Associativity of composition

associative-comp-hom-species :
  {l1 l2 l3 l4 l5 : Level} {F : species l1 l2} {G : species l1 l3}
  {H : species l1 l4} {K : species l1 l5}
  (h : hom-species H K) (g : hom-species G H) (f : hom-species F G) ‚Üí
  Id ( comp-hom-species (comp-hom-species h g) f)
     ( comp-hom-species h (comp-hom-species g f))
associative-comp-hom-species h g f = refl
