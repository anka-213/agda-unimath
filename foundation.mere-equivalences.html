Idea

Two types X and Y are said to be merely equivalent, if there exists an equivalence from X to Y. Propositional truncations are used to express mere equivalence.

Definition

mere-equiv-Prop :
  {l1 l2 : Level} ‚Üí UU l1 ‚Üí UU l2 ‚Üí UU-Prop (l1 ‚äî l2)
mere-equiv-Prop X Y = trunc-Prop (X ‚âÉ Y)

mere-equiv :
  {l1 l2 : Level} ‚Üí UU l1 ‚Üí UU l2 ‚Üí UU (l1 ‚äî l2)
mere-equiv X Y = type-Prop (mere-equiv-Prop X Y)

abstract
  is-prop-mere-equiv :
    {l1 l2 : Level} (X : UU l1) (Y : UU l2) ‚Üí is-prop (mere-equiv X Y)
  is-prop-mere-equiv X Y = is-prop-type-Prop (mere-equiv-Prop X Y)

Properties

Mere equivalence is reflexive

abstract
  refl-mere-equiv :
    {l1 : Level} (X : UU l1) ‚Üí mere-equiv X X
  refl-mere-equiv X = unit-trunc-Prop id-equiv

Mere equivalence is symmetric

abstract
  symmetric-mere-equiv :
    {l1 l2 : Level} {X : UU l1} {Y : UU l2} ‚Üí mere-equiv X Y ‚Üí mere-equiv Y X
  symmetric-mere-equiv {l1} {l2} {X} {Y} =
    map-universal-property-trunc-Prop
      ( mere-equiv-Prop Y X)
      ( Œª e ‚Üí unit-trunc-Prop (inv-equiv e))

Mere equivalence is transitive

abstract
  transitive-mere-equiv :
    {l1 l2 l3 : Level} {X : UU l1} {Y : UU l2} {Z : UU l3} ‚Üí
    mere-equiv X Y ‚Üí mere-equiv Y Z ‚Üí mere-equiv X Z
  transitive-mere-equiv {X = X} {Y} {Z} e f =
    apply-universal-property-trunc-Prop e
      ( mere-equiv-Prop X Z)
      ( Œª e' ‚Üí
        apply-universal-property-trunc-Prop f
          ( mere-equiv-Prop X Z)
          ( Œª f' ‚Üí unit-trunc-Prop (f' ‚àòe e')))
